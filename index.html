JavaScript Functions

To make our lives easier I have made a few functions that you are welcome to copy and paste into your own projects. These functions are maliable and not only save you time but are a good example of how knowing a little javascript can go a long way. Think of them as tools in a Utility Belt and apply them at the start of your project and use them as often as you like.

There are examples of use cases in the component files. Please try them out and let me know if you have any questions or issues.
